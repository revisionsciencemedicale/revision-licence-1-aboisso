Révision Licence 1 - Composition en ligne

Fichiers :
- index.html : page principale
- style.css : apparence
- script.js : logique de l'application

Nouveauté ajoutée :
Les sujets peuvent être enregistrés en mémoire sans apparaître sur la page d'accueil.
Seuls les sujets marqués "Programmé" dans l'espace ADMIN s'affichent aux étudiants.

Utilisation :
1. Ouvre index.html dans un navigateur.
2. Clique sur Administration.
3. Mot de passe par défaut : admin123
4. Ajoute plusieurs sujets.
5. Clique sur "Programmer" pour afficher un sujet à l'accueil étudiant.
6. Clique sur "Retirer" pour garder le sujet en mémoire sans l'afficher.

Important :
Les données sont sauvegardées dans le navigateur avec localStorage.

Modification : les étudiants peuvent composer/recomposer le même sujet plusieurs fois avec le même matricule.
